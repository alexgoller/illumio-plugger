Build Your Own Plugin

Scaffold a new plugin in Go, Shell, or Python with the Illumio SDK pre-configured. Your plugin runs as a Docker container with automatic credential injection.

plugger create my-plugin -t go plugger create my-plugin -t shell plugger create my-plugin -t python
